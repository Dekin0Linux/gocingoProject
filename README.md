# Gocingo

Gocingo is a Next.js application designed to showcase various projects and connect donors with those in need. This project demonstrates the use of Next.js for server-side rendering, API data fetching, and dynamic content rendering.

## Project Description

Gocingo aims to provide a platform where users can view different charitable projects. Each project includes details such as title, description, and an image. The application fetches project data from an API and displays it in a responsive grid layout.
